## How Videos Are Used

The videos are loaded in `final.html` via HTML `<video>` tags:
- Prologue plays on page load, then auto-transitions to the game
- Ending A plays when the player wins Stage 2
- Ending B plays when both players run out of lives

See `storyboard/` for the written storyboard and `storyboard/frames/` for first/last frame images of each clip.
